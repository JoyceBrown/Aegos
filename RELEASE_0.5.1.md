# Aegos 0.5.1

## 更新重点
- 删除首页当前节点下方的端口/复制行，减少顶部信息噪声。
- 调整连接区纵向分布，缓解控件挤在上方、下方留白过多的问题。
- 推荐区改为“推荐节点 香港”，推荐地区由最低延迟可用节点决定，说明文字保留“延迟最低·稳定性高”。
- 常用地区筛选移入常用节点卡片顶部。
- 快捷操作扩展为两行四列，补充连接管理和订阅管理入口。
- 重新分配底部状态指标列宽，避免最小横向窗口下代理端口被隐藏，同时收窄上传/下载占位。
- 为标题栏和品牌区加入 Tauri 原生拖动区域，并保留 JS 拖动兜底。
- 降低状态轮询和控制器请求阻塞时间，减少点击反应慢和界面卡顿体感。

## 验证
- `node --check src/app.js`
- `npm run smoke:ui`
- `npm run check`
- `npm run build`
- `npm run audit:release`
