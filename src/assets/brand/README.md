# Aegos brand mark

The Aegos mark combines the Microsoft Fluent UI System Icons `Shield 48 Filled` silhouette with an original A-shaped routing path and Aegos color treatment.

- Fluent source commit: `9a1129bb2432b163b48044341664c68a3c100908`
- Upstream license: MIT, preserved under `third_party/fluent-ui-system-icons/LICENSE`
- Canonical application source: `aegos-mark.svg`
- Sidebar raster: `aegos-mark-64.png`, generated from the canonical SVG to avoid runtime SVG parsing during startup
- Windows application and installer icon: generated from this SVG

Do not maintain a separate taskbar or installer logo.
